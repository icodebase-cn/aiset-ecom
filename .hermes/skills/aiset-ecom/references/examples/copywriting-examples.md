# 电商视觉文案示例库

每个示例展示完整工作流：收集信息 → 提炼合规版必卖理由 → 确认 → 生成主图 + 详情页 → 自审评分。
示例仅作参考，不构成法律意见或平台审核承诺。

---

## 示例 1：普通食品

### 输入

```text
产品类型：普通食品，非保健食品，非药品
品牌：熊花木末
产品：松花粉固体饮料
成分：松花粉、麦芽糊精、黄精、山楂、薄荷
规格：60g/盒
目标用户：希望日常饮用方便饮品的上班族
主图初稿：突出速溶、小条便携、口感清爽、随身携带
证据：暂无保健食品批准文件
平台：淘宝 / 抖音小店
```

### 合规版必卖理由

```text
1. 独立小条包装：方便办公室、通勤、出行随身携带。
2. 速溶冲饮体验：适合日常饮品场景，无需复杂冲泡。
3. 松花粉+植物成分：可描述原料和风味，不得暗示保健功效。
4. 普通食品边界：必须明确标注非保健食品、非药品。
```

### 预期输出

```text
[图1 / 首图]

画面：
产品盒和独立小条居中，旁边放一杯冲调后的饮品。背景用清爽绿金色，突出原料和日常饮用场景。

图内文案：
标题：松花粉固体饮料
副标题：小条装 随身带
卖点：速溶 · 清爽 · 日常饮用
声明：普通食品 非保健食品

设计说明：
主标题占视觉中心，声明放底部但保持可读。不得出现免疫、调理、改善睡眠、养生功效等暗示。
```

## 示例 2：运动器材

### 输入

```text
产品类型：健身器材，非医疗器械
产品：体态训练弹力带
目标用户：久坐上班族，希望在家做体态训练
主图初稿：改善体态、修复驼背、缓解腰背痛
证据：无医疗器械证书
平台：京东 / 抖音小店
```

### 高风险初稿

```text
修复驼背，缓解腰背痛，矫正脊柱问题。
```

### 合规改写方向

```text
居家体态训练，肩背拉伸支撑，轻量弹力辅助。
```

### 预期输出

```text
[图2 / 使用场景]

画面：
三场景分镜：办公室休息拉伸、居家训练、运动前热身。

图内文案：
标题：日常体态训练
副标题：肩背拉伸辅助
卖点：轻量 · 便携 · 易上手
声明：健身器材 非医疗产品

设计说明：
避免"治疗、矫正、脊柱病、腰背痛缓解、康复"等医疗化用词，改用"训练、锻炼"语言体系。
```

## 示例 3：合规审查

### 输入

```text
产品类型：普通食品
文案初稿：
100%有效改善睡眠。最好的天然配方。坚持喝7天恢复元气、提升免疫力。
证据：无检测报告、无保健食品批文、无临床依据
```

### 审查结果

```text
高风险。初稿存在多处问题：

1. "100%有效"：绝对化且无证据支撑。
2. "最好"：绝对化营销宣称。
3. "改善睡眠""提升免疫力"：保健功效宣称，普通食品严禁。
4. "坚持喝7天"：无依据的效果承诺。
5. 普通食品不得暗示治疗、保健改善或身体功能变化。
```

### 合规改写方向

```text
产品名：植物固体饮料
可用卖点：成分组合、口感清淡、小条便携、日常饮用场景
必须声明：普通食品，非保健食品，非药品
```

### 预期输出

```text
[合规改写版]

画面：
原料平铺图搭配产品包装和冲调后的饮品。

图内文案：
标题：植物固体饮料
副标题：清淡口感 随身便携
卖点：小条装 · 日常饮用 · 清爽风味
声明：普通食品 非保健食品 非药品

设计说明：
移除所有保健功效宣称。视觉重心放在原料、口感、包装和日常饮用场景上。
```

## 自审清单（新增示例前必过）

- 证据和资质状态已说明。
- 高风险词已标出（如有）。
- 输出遵循"画面 / 图内文案 / 设计说明"三项结构。
- 主图文案足够简短，可放进真实移动端电商图片。
- 敏感品类已保留必要免责声明。
- 示例不作为最终合规保证或法律意见。
